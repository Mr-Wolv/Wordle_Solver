HOW TO PLAY

  1. Type your guess. Just press the letter keys — no box to click. The word appears on the board's active row (max 5 letters).
  2. Fix a tile's color. After you type, click each tile on the active row to cycle it absent → present → correct — matching the colors from your real Wordle.
  3. Submit. Press Enter or the on-screen ENTER key. The board's top suggestion is your optimal next word — play it.
  4. Delete. Press Backspace or the on-screen DELETE key to remove the last letter typed.
  5. NYT hint (optional). Type one consonant + one vowel in the hint box and log them — the solver still closes 100% with no hints, so skip it if you like.
  6. Done? Use SYSTEM RESET for a new puzzle. EXIT APP quits.

The game is fully solvable with no hints. The NYT hint (1 consonant + 1 vowel) is an optional shortcut, not required.
